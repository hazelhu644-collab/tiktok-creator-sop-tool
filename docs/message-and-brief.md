# Message and Brief Rules

## Goal

This file defines how the tool should generate creator messages and product briefs.

The first MVP focuses on TikTok Shop creator collaboration for a pet steam grooming brush.

## Message Output Language

When generating a message, always show:

1. English message first
2. Chinese explanation below

The Chinese explanation should explain the purpose of the message and why it is written that way.

## Message Tone

The English message should sound like a US brand communicating with a creator.

Tone requirements:

* Professional but warm
* Clear and direct
* Not too humble
* Not overly excited
* Not pushy
* Not like a translated Chinese message
* No excessive compliments
* No begging language
* Suitable for TikTok Shop creator collaboration

Avoid phrases like:

* Please please
* We really really hope
* You are amazing and perfect
* We are begging you
* Sorry to bother you so much

Preferred style:

* Thanks for the update.
* Just checking in.
* Could you focus more on...
* This helps viewers understand the product better.
* Please let us know your expected posting date.
* If you are no longer able to move forward, please let us know so we can update the campaign status.

## Channel Rules

### TikTok DM

Use short and natural messages.

Length:

* 2 to 4 short sentences

Style:

* Conversational
* Direct
* Easy to read
* No long paragraphs

### TikTok Shop Affiliate Message

Use a medium-length message.

Length:

* 4 to 6 sentences

Style:

* Clear
* Professional
* Includes collaboration context
* Includes next action

### Email

Use a more structured message.

Structure:

* Greeting
* Short context
* Clear request
* Next step
* Closing

### WhatsApp

Use a conversational message, slightly more detailed than TikTok DM.

## Product Brief: Steam Grooming Brush

### Product Name

Steam Grooming Brush

### Video Requirements

Each creator should deliver:

* 2 videos
* Each video should be 60 seconds or longer
* Must tag the brand account
* Must add the TikTok Shop product link

### Video 1 Direction

Real review style.

Goal:

Show that the product actually removes loose hair.

Must include:

1. Show the pet shedding problem or show the loose hair collected
2. Show the mist feature clearly
3. Groom the pet in a real scene
4. Show the pet’s real reaction
5. Show the loose hair removed
6. Show quick cleanup

### Video 2 Direction

Daily pet-care routine style.

Goal:

Make viewers feel this product fits into their daily pet-care routine.

Must include:

1. Real home setting
2. Pet-care routine moment
3. Mist feature
4. Pet looking comfortable or not resisting
5. Loose hair removed
6. Easy cleanup

### Content Priority

The most important shots are:

1. Loose hair removed
2. Mist feature
3. Pet’s real reaction
4. Natural daily pet-care scene
5. Easy cleanup

### What Creators Should Avoid

Creators should avoid:

* Only filming unboxing
* Only filming the product on a table
* Not showing the pet using the product
* Not showing the mist
* Not showing the loose hair collected
* Making the video look like a hard ad
* Forgetting to tag the brand account
* Forgetting to add the product link

## Follow-up Scenarios

The tool should support these message scenarios:

### Light Follow-up

Used when:

* Creator was contacted 2 days ago
* No reply yet
* No sample has been sent

Goal:

Check whether the creator is interested.

### Sample Delivered Follow-up

Used when:

* Sample has been delivered
* No video has been posted yet

Goal:

Remind the creator to film based on the brief.

### Second Follow-up

Used when:

* The seller already followed up
* Creator still has not replied

Goal:

Ask for a clear update.

### Second Video Reminder

Used when:

* Creator posted 1 video
* Collaboration requires 2 videos

Goal:

Ask the creator to post the second video.

### Final Follow-up Before Failed Candidate

Used when:

* Sample was already sent or delivered
* Creator has not replied after multiple follow-ups
* Collaboration may need to be marked as failed

Goal:

Ask for one final update before the seller updates the campaign status.
